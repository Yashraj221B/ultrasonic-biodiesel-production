# Ultrasonic Biodiesel Production

Preparation of Biodiesel using Ultrasonic Cavitation Technology with interactive process simulation and animations.

## 🚀 Features
- Ultrasonic biodiesel production simulation
- Step-by-step process visualization
- Interactive UI with animations
- Lab-scale process representation

## 🛠 Tech Stack
- React + Vite
- Tailwind CSS
- Framer Motion
